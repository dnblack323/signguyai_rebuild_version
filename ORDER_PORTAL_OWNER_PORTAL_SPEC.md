# Order Portal Owner Portal Specification

## Purpose

The Store Owner Portal gives the sign shop's customer a simple place to complete setup, review the launch packet, approve the portal, share the portal, and track performance.

## Screens

- Dashboard
- Questionnaire
- Upload Artwork
- Store Launch Packet
- Store Preview
- Products
- QR Code / Share Link
- Promotional Copy
- Sales Progress
- Fundraiser Progress
- Donations
- Payouts
- Messages
- Reports

## Owner Actions

Portal owner can:

- answer questionnaire
- upload logo/artwork/files
- select known products
- choose "open to suggestions"
- review AI-prepared setup summary where exposed
- review product mockups
- request changes
- approve launch packet
- approve store
- complete Stripe payout onboarding if needed
- view deadline/status/progress
- view sales/fundraiser/donation totals
- view estimated payout
- copy share link
- download QR code
- copy promotional text
- view final reports

## Privacy Rules

Portal owner must not see:

- shop production cost
- shop margin
- internal shop notes
- internal template cost assumptions
- supplier costs
- platform admin data
- unrelated tenant/store data
- internal production problems unless intentionally shared

## Approval Rules

Owner approval records:

- approver name/email
- approved packet/store snapshot
- approved terms/fee summary
- approval timestamp
- IP/user agent where available
- change requests if not approved

Approval is required before launch.
